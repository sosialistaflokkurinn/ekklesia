## Overview

This folder contains:
- [Security policies](policies/) and migration plans
- [Active security research](current/) and work logs
- [Historical security incidents](historical/) and responses (organized by date)
- [GitHub issue-specific security reviews](responses/)

See [INDEX.md](INDEX.md) for detailed security overview and [incident response procedures](https://owasp.org/www-community/Incident_Response).
